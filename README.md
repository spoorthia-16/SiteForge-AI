# SiteForge-AI Website Builder

AI Website Builder is a full-stack web application that enables users to generate professional and responsive websites using Artificial Intelligence. Users can simply describe their website requirements through a prompt, and the AI automatically generates website content, structure, and design suggestions, significantly reducing development time and effort.

The platform provides an intuitive interface for creating, customizing, previewing, and managing websites without requiring extensive coding knowledge. It combines modern web technologies with AI-powered content generation to help individuals, startups, and businesses quickly build their online presence.

# Features

🤖 AI-powered website generation from user prompts

🎨 Automatic content and layout creation

📱 Fully responsive website designs

🔐 Secure user authentication and session management

💾 Project saving and management

⚡ Real-time website preview

🖥️ Modern and user-friendly dashboard

🌐 Fast and scalable full-stack architecture


# Tech Stack

Frontend: React.js, TypeScript, Tailwind CSS, ShadCN UI

Backend: Node.js, Express.js

Database: PostgreSQL / Prisma ORM

Authentication: JWT / Session-based Authentication

AI Integration: LLM-based website content generation
